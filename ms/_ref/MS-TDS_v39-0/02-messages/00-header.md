# Messages

The formal syntax of all messages is provided in Augmented Backus-Naur
Form (ABNF)
[\[RFC4234\]](https://go.microsoft.com/fwlink/?LinkId=90462), with the
addition of the following:

-   underscore (\"\_\") -- a valid character within an identifier.

-   \"%x00\" -- a valid value.

-   \"%b0\" -- a valid value.
